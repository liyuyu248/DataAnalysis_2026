# GitHub Upload Steps

## Option A: Upload with Git command line

Open Terminal / Git Bash inside the `DataAnalysis_2026` folder and run:

```bash
git init
git add .
git commit -m "Add course modules and homework results"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/DataAnalysis_2026.git
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username.

## Option B: Upload in GitHub web interface

1. Create a new public repository named `DataAnalysis_2026`
2. Click **Add file** -> **Upload files**
3. Upload the full folder structure
4. Commit changes
